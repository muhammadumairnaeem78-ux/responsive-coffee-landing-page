# Mocha Mousse Roasters ☕

A lightweight, fully responsive, and accessible landing page built strictly with vanilla web technologies. This project was developed to demonstrate a strong mastery of frontend fundamentals without relying on external frameworks or libraries.

## 🚀 Live Demo
*(Once you enable GitHub pages, paste your live link here!)*
[View Live Site](https://muhammadumairnaeem78-ux.github.io/responsive-coffee-landing-page)

## ✨ Features
* **Mobile-First Responsive Design:** Adapts seamlessly to mobile, tablet, and desktop screens.
* **Semantic HTML5:** Structured using proper landmarks (`<header>`, `<nav>`, `<main>`, `<article>`, `<footer>`) for superior SEO and accessibility rather than generic `<div>` wrappers.
* **Modern CSS Architecture:** Built using **CSS Grid** for macro-layouts and **Flexbox** for micro-components.
* **Fluid Typography:** Utilizes CSS `clamp()` to mathematically scale font sizes across viewport changes without sudden media query jumps.
* **Vanilla JavaScript Logic:** Includes a custom-built mobile menu toggle and client-side form validation (Regex email checking).
* **Accessibility (WCAG AA):** Includes proper ARIA attributes (`aria-expanded`, `aria-live`, `aria-invalid`), semantic inputs, and high-contrast color palettes for screen readers and keyboard navigation.

## 🛠️ Tech Stack
* **HTML5** * **CSS3** (Variables, Grid, Flexbox, Media Queries)
* **Vanilla JavaScript** (ES6+)
* **Fonts:** Inter & Roboto (via Google Fonts)
